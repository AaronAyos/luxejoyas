Landing page de joyería — Instrucciones

Cómo ver localmente:

1. Abre `Landing/index.html` en tu navegador.
2. Reemplaza las imágenes dentro de `Landing/assets/` por tus fotos.
3. Para integración con CRM, conecta el formulario (`script.js`) a tu endpoint.

Archivos:
- index.html — HTML del landing
- style.css — Estilos responsivos
- script.js — Manejo básico del formulario

Notas:
- Puedes ajustar colores en `style.css`.
- Si quieres, puedo generar versiones con fotos reales y variantes de texto.
